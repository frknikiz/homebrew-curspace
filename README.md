# homebrew-curspace

Homebrew tap for [Curspace](https://github.com/frknikiz/curspace) — a terminal-first project discovery and workspace launcher for Cursor IDE.

## Install

```bash
brew tap frknikiz/curspace
brew install curspace
```

## Update

```bash
brew update
brew upgrade curspace
```

## Uninstall

```bash
brew uninstall curspace
brew untap frknikiz/curspace
```
